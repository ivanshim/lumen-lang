================================================================================
LUMEN-LANG: LANGUAGE-AGNOSTIC FRAMEWORK ARCHITECTURE
================================================================================

This document describes the directory structure for the language-agnostic
interpreter framework and language implementations.

================================================================================
DIRECTORY STRUCTURE
================================================================================

lumen-lang/
│
├── src/                                 # Binary entry point and dispatcher
│   └── main.rs                          # Kernel selection and binary delegation
│
├── src_stream/                          # STREAM (procedural, AST-based) KERNEL
│   ├── main.rs                          # Stream kernel binary entry point
│   ├── kernel/                          # Language-agnostic kernel core
│   │   ├── lexer.rs                     # Tokenization infrastructure
│   │   ├── parser.rs                    # Generic AST parsing engine
│   │   ├── registry.rs                  # Token and handler registration
│   │   ├── eval.rs                      # Tree-walking interpreter
│   │   └── runtime/                     # Runtime value primitives
│   └── languages/                       # Language-specific implementations
│       ├── lumen/                       # Lumen language (indentation-based)
│       │   ├── expressions/
│       │   ├── statements/
│       │   ├── structure/
│       │   ├── extern_system/
│       │   └── dispatcher/
│       ├── mini_rust/                   # Mini-Rust language (brace-based)
│       │   ├── expressions/
│       │   ├── statements/
│       │   ├── structure/
│       │   └── dispatcher/
│       └── mini_python/                 # Mini-Python language (indentation-based)
│           ├── expressions/
│           ├── statements/
│           ├── structure/
│           └── dispatcher/
│
├── src_microcode/                       # MICROCODE (data-driven, table-based) KERNEL
│   ├── main.rs                          # Microcode kernel binary entry point
│   ├── kernel/                          # Data-driven execution pipeline
│   │   ├── ingest.rs                    # Schema parsing and validation
│   │   ├── reduce.rs                    # Instruction compilation
│   │   ├── execute.rs                   # Instruction interpretation
│   │   └── primitives.rs                # Primitive operation definitions
│   ├── runtime/                         # Runtime support and external calls
│   │   └── extern/                      # External function dispatch
│   ├── languages/                       # Language schema definitions
│   │   ├── lumen/
│   │   ├── mini_rust/
│   │   ├── mini_python/
│   │   └── alien_lang/
│   └── schema/                          # Schema structure and validation
│
├── examples/                            # Sample programs (lumen, mini-rust, mini-python)
├── archive/                             # Inactive or experimental language tracks
├── README_LANGUAGES.md                  # Cross-language design notes
├── MICROCODE_ARCHITECTURE.md            # Microcode kernel overview
├── VERSION_HISTORY.md                   # Release notes and changelog
├── Cargo.toml
├── README.md                            # Project overview
└── DIRECTORY_STRUCTURE.txt              # This file
